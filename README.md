# ama_moderator

Ask questions and stuff.

## Required

- DO Buoy Gem

### Google Oauth2 Credentials

ENV variables required for google-oauth2 omniauth:

- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`

Google+ must be enabled.
